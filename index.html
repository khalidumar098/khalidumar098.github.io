Parking Garage System

Parking Mania

Click for live demo.

People spend a lot of time in a parking garage looking for an available space to park. Furthermore, high volume of cars needing parking in peak hours creates congestion's in parking garages, resulting in more time wasted looking for an available spot. We came up with the solution to this problem by implement a system that allows customers to reserve and view available spaces in parking garage. We designed and developed our website using HTML, CSS, Java Script, MySQL and PHP. Customers will be able to reserve spots for a specified check in time. There will be short term and long-term parking reservations which will allow customers with a variety of needs to utilize the parking garage. There will be different plans available for customers like monthly, daily and hourly basis. Also, users can contact us anytime if they have any questions through our website easily in a Contact Us section. Admin can maintain daily transactions and keep track of the bookings easily and more effectively.

  • Users can Sign-up/Sign-in
  • Reservation - to save a spot reservation online ahead of time
  • Cancel reservation - to cancel a reservation before parking
  • Track Time- to track how much time has elapsed since parking
  • Admin can view the number of vehicle entries in a particular period
  • Admin can add vehicle which is going to park in garage and add parking charges
  • Admin can maintain daily transactions and manage bookings
  • Admin can manage incoming and outgoing vehicles
  • Can view graphical interface of parking garage.

Image Editor

  • Pillow is an image processing library that can be used to manipulate images quite easy. PIL has not seen any development since 2011. Pillow is a fork of PIL (Python Image Library), started and maintained by Alex Clark and Contributors. It was based on the PIL code, and then evolved to a better, modern and more friendly version of PIL. It adds support for opening, manipulating, and saving many different image file formats. It supports a range of image file formats such as PNG, JPEG, PPM, GIF, TIFF, BMP and so on.
  • In this project, first I checked whether the library is installed or not by using command pip list. User can see the menu with multiple options to select the operation which they want to perform. I used standard HD image to use it for my project and the image saves it as 1920 x 1080 dimension. The required image should be in the same directory as the python script file being run. User can select their own image by given image address in Image.open(path). User can perform various operations like see an image, resizing, color transform, cropping, flipping, image details, rotating and so on. All the new files will be created in the same folder where the original image is after successfully execution. User can also see about me section and exit the program through the menu. Users can get benefits without need to pay for expensive editor software’s by using simple and free pillow library.
  • By using Pillow, we can perform most common image processing operations found in applications opening, manipulating and saving many different image file formats. We can do much more with the pillow package like image transforms, processing bands, image enhancement, the ability to print the images and much more.
